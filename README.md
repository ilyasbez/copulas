# Copulas – Minimierung des Drawdowns eines Aktienportfolios

Dieses Projekt entstand im Rahmen meiner Bachelorarbeit und untersucht den Einsatz von **Copulas** zur Modellierung von Abhängigkeitsstrukturen zwischen Aktienrenditen mit dem Ziel, den **Maximum Drawdown** eines Portfolios zu minimieren.

---

## Projektbeschreibung

Im Fokus der Analyse stehen die Aktien von **Apple (AAPL)** und **Amazon (AMZN)**. Mithilfe von Copula-Modellen werden die gemeinsamen Verteilungen der Renditen erfasst, um Portfoliogewichte zu bestimmen, die das Drawdown-Risiko reduzieren.

# Copulas – Minimizing Portfolio Drawdown

This project was developed as part of my **bachelor thesis** and explores the use of **copulas** to model dependency structures between stock returns, with the goal of minimizing the **maximum drawdown** of a portfolio.

---

## Project Description

The analysis focuses on **Apple (AAPL)** and **Amazon (AMZN)** stocks. Copula models are employed to capture the joint distribution of returns and to determine portfolio weights that reduce drawdown risk.

---

## Contents

| File | Description |
|------|-------------|
| `Copula_AAPL_AMZN.html` |  Presentation of results and visualizations as a Notebook |
| `src/` | Source code for data processing, copula estimation, and optimization |

---

## Outlook

Extensions to **larger portfolios** with multiple assets are planned to evaluate the scalability and practical applicability of this approach.
