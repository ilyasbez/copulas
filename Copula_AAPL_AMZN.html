Die t Copula hat mit 271.8 den höchsten loglikelihood Wert mit Parametern df=3.751 und rho=0.56. Dennoch wollen wir den Unterschied der Copulas graphisch vergleichen. Dazu ziehen wir zunächst uniforme Samples aus den Copulas

n = nrow(U)

Ug = rCopula(n, fit_gauss@copula)
Ut = rCopula(n, fit_t@copula)
Uc = rCopula(n, fit_clayton@copula)
Uf = rCopula(n, fit_frank@copula)

Diese werden wieder auf ihre t-marginals zurücktransformiert.

inv_aapl = function(u) {
  mean_t_aapl + sd_t_aapl*qt(u, df=df_t_aapl)
}
inv_amzn = function(u) {
  mean_t_amzn + sd_t_amzn*qt(u, df=df_t_amzn)
}

Die modellierte Verteilungsfunktion kann jetzt genutzt werden, um ein Portfolio aus AAPL und AMZN zu bauen, dass den CVaR minimiert.

Wir samplen mittels einer Monte-Carlo-Simulation verschiedene Log-Renditen Paare und berechnen aus diesen Werten den (empirschen) VaR und CVaR.

n_sim = 100000

U_sim = rCopula(
  n_sim, 
  fit_t@copula
)

sim_aapl = inv_aapl(U_sim[,1])
sim_amzn = inv_amzn(U_sim[,2])

sim_returns = cbind(sim_aapl, sim_amzn)

colnames(sim_returns) = c("AAPL", "AMZN")

Die CVaR Funktion der Verluste ist definiert durch

cvar_function = function(port_return_losses, alpha=0.95) {
  #alpha-Quantil, an dem Verluste VaR übersteigen (5% höchste Verluste)
  #VaR
  var_alpha = quantile(
    port_return_losses, 
    probs = alpha
  )
  
  #Durchschnitt der Verluste über VaR
  cvar = mean(port_return_losses[port_return_losses >= var_alpha])
  
  return(cvar)
}

Das Gesamtportfolio (bzw. seine LogRenditen) approximieren wir durch R_P = wR_AAPL + (1-w)R_AMZN für w in [0,1]. Es genügt also in einer Variable zu minimieren.

# Zielfunktion 
objective_function = function(w_aapl) {
  # Nebenbedingungen
  if(w_aapl < 0 || w_aapl > 1){
    return(1e6)
  }
  
  w_amzn = 1- w_aapl
  
  pf_returns = w_aapl*sim_returns[,1]+w_amzn*sim_returns[,2]
  pf_losses = -pf_returns
  
  cvar = cvar_function(pf_losses, alpha=0.95)
  
  return(cvar)
}

Numerische Optimierung liefert nun

opt_result = optimize(
  f=objective_function,
  interval=c(0,1)
)
